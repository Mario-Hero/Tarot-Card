# Tarot-Card

**Last Update: 2021.05.02**

塔罗牌抽牌Python脚本




## 用法 Usage

随意在键盘上输入，按下回车，程序会根据输入的值给出抽牌结果并显示图片。

请下载名为tarot的图片文件夹，并与该python脚本放在同一文件夹。（或者直接下载整个项目即可）



## 需求 Needs

Python 3

安装 matplotlib



## 参数 Parameters

SHOW_IMAGE = True 是否显示图片
